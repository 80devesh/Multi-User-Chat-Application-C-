# Multi-User Chat Application

A simple multi-user chat application built in C# using .NET, consisting of a server and multiple clients that communicate via TCP sockets. The server handles multiple connections and routes messages between connected clients.

## Features

- Supports multiple clients connecting to the server.
- Real-time messaging between connected clients.
- Basic command-line interface for interaction.

## Getting Started

### Prerequisites

- [Visual Studio](https://visualstudio.microsoft.com/downloads/) or any compatible IDE
- .NET SDK (version 6 or later)
